Rubik-Bot

A Rubik-Bot egy általam fejlesztett Discord bot, amelyet Python programozási nyelven írtam. Ez a bot számos interaktív funkcióval rendelkezik, mint például a kő-papír-olló játék. A Rubik-Bot egyszerű parancsokkal vezérelhető, amelyeket a felhasználók a Discord csatornákon keresztül adhatnak meg. A bot azonnali visszajelzést ad, és egy mappában naplózza az összes aznapi parancslekérést, megjelölve, hogy melyik szerveren, melyik csatornán, mikor és ki küldte be a parancsot.

A kő-papír-olló játék kódja:

                    
@bot.command()
async def rps(ctx):
    choices = ['rock', 'paper', 'scissors']
    bot_choice = random.choice(choices)
                            
    message = await ctx.send("Choose one: rock, paper or scissors")
                            
    def check(m):
        return m.author == ctx.author and m.content.lower() in choices
                            
    try:
        player_choice = await bot.wait_for('message', timeout=60.0, check=check)
    except TimeoutError:
        await ctx.send("You didn't reply in time.")
        return
                            
    player_choice = player_choice.content.lower()
                                
    if player_choice == bot_choice:
        await ctx.send(f'Tie! You both chose: {player_choice}.')
    elif (player_choice == 'rock' and bot_choice == 'scissors') or \
         (player_choice == 'paper' and bot_choice == 'rock') or \
         (player_choice == 'scissors' and bot_choice == 'paper'):
        await ctx.send(f'Congratulations, you won!')
        await ctx.send(f'You chose: {player_choice'})
        await ctx.send(f'The bot chose: {bot_choice}')
    else:
        await ctx.send(f'Sorry, you lost!')
        await ctx.send(f'You chose: {player_choice})
        await ctx.send(f'The bot chose: {bot_choice}')

Így néz ki Discordon:

A Rubik-Bot emellett számos hasznos eszközt is biztosít a Discord szerverek adminisztrátorainak. Automatizált üdvözlő üzenetekkel fogadja az új tagokat, moderációs eszközöket kínál a szabálysértések kezelésére, és különféle értesítéseket küldhet a szerver tagjainak. A bot integrálható különböző API-kkal, így például valós idejű időjárás-jelentéseket vagy egyéb külső adatokat is szolgáltathat.

Az Időjárás lekérő parancs kódja:

                    
@bot.command()
async def weather(ctx, *, location):
    params = {'q': location, 'appid': API_KEY, 'units': 'metric'}
    response = requests.get(BASE_URL, params=params)
    data = response.json()
                                
    if response.status_code == 200:
        weather_description = data['weather'][0]['description']
        temperature = data['main']['temp']
        humidity = data['main']['humidity']
        wind_speed = data['wind']['speed']
                                    
        await ctx.send(f'The weather in {location}:')
        await ctx.send(f'{weather_description}')
        await ctx.send(f'Temperature: {temperature}°C')
        await ctx.send(f'Humidity: {humidity}%')
        await ctx.send(f'Wind speed: {wind_speed} m/s')
    else:
        await ctx.send("Couldn't not query the weather for this city.")
        await ctx.send(f'Please try again')
